# Projeto_Sprint5
 Dashboard de Análise de Anúncios de Carros

Este aplicativo web foi desenvolvido para análise interativa de dados de anúncios de vendas de carros. 

## Funcionalidades:
- Visualização de histogramas para análise de distribuição de dados
- Gráficos de dispersão para identificar correlações entre variáveis
- Interface interativa com controles para personalizar as visualizações

## Objetivo:
Facilitar a análise de dados de vendas de carros através de visualizações interativas, permitindo insights sobre preços, quilometragem e outras características dos veículos anunciados.
